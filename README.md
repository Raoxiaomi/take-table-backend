## 项目介绍

该Web后端项目采用 nestjs + typeorm + mysql80 开发。主要用于将收集的信息通过该后端写进数据库中。

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
